Web.config
	1) PocketPath пустой => файлы сайта-образца не кэшируются (режим прокси)

url.config
	1) <MessageFrom></MessageFrom>
	   <MessageTo></MessageTo>
	   настройки для отправки форм
	2) <Locked>true</Locked>
	   => запрещает обращаться за файлами к образцу и записывать/переписывать файлы
 	   <Locked>1999-12-01Т00:00:00</Locked> (прим. следить, что T латинская)
           => то же с некой даты
	3) <Replacement>
		4а) target = {[пусто]|относительный путь}
		    => замена во всех сэмплах, либо только в конкретном

/Sheets
	во все тексты кроме css и скриптов внедряется текст из файлов этой папки
/Views/Shared/Controls
	все контролы из этой папки рэндерятся с передачей модели из контроллера (по умолчанию передается Pocket) и заменяют собой строки в формате @ИмяКонтрола.chtml
/url.log
	если файл существует, в него пишется лог
	
Адресная строка браузера
	1) #admin переводит в режим администрирования
	2) #reset обнуляет текущую модель
	3) #switch=[url] переключение текущего покета
