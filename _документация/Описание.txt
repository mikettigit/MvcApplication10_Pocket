Web.config
	1) PocketSource пустой => при старте открывается индексная страница (режим сервиса по копированию)
	2) PocketPath пустой => файлы сайта-образца не кэшируются (режим прокси)

url.config
	1) Пустой id
		<Configuration id="00000000-0000-0000-0000-000000000000">
	   => не подключаются элементы из Sheets/ForTrials (диалоги редактирования)
	2) <MessageFrom></MessageFrom>
	   <MessageTo></MessageTo>
	   настройки для отправки форм

после изменения url.config достаточно удалить файлы с hash, чтобы не переполучать исходники от сайта-образца (не нужно переполучать, т.к. образец уже может быть сильно перерисован)
