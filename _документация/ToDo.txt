Функционал
1) есть проблема при обновлении платформы - нужно удаление текстов со старыми hash, возможно ввести версии и при несоответствии версии в config пересобирать как-то все тексты
2) cамообучение для закрытия всплывающих окон - м.б. DOM Mutation Observer http://stackoverflow.com/questions/2844565/is-there-a-javascript-jquery-dom-change-listener
3) самообучение при работе с корзиной - возможно прощелкивание всех ссылок и кнопок
4) механизм вырезания блока по открывающему тэгу вместо display:none



Рефакторинг
1) PocketController.GetConfig должен быть по Get, а не Post
2) post.js - ajaxSuccess не должен отрабатывать служебные URL - например "/pocket/getconfig" - без их указания в тексте
3) RouteConfig - не нужно прописывать каждый служебный URL
